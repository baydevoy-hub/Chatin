# Chatin Project

## Project Description
Chatin is a real-time chat application that allows users to communicate seamlessly. With features like private messaging, group chats, and message notifications, Chatin provides a user-friendly experience for everyone.

## Prerequisites
- Node.js (version 12 or higher)
- npm (Node Package Manager)
- MongoDB server setup
- Git for version control

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/baydevoy-hub/Chatin.git
   ```
2. Navigate into the project directory:
   ```bash
   cd Chatin
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Set up your MongoDB connection in the `.env` file.
5. Run the application:
   ```bash
   npm start
   ```
6. Open your browser and visit `http://localhost:3000` to access the app.
